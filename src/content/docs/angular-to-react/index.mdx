# Angular to React 마이그레이션 가이드

이 eBook은 Angular 애플리케이션을 React로 마이그레이션하는 전체 과정을 단계별로 안내합니다.

## 📚 목차 구성

이 가이드는 5개 파트로 구성되어 있습니다:

### 1부: 기초 개념
Angular의 핵심 개념을 React로 변환하는 기초를 다룹니다.
- 프로젝트 설정 및 구조
- 컴포넌트 기본 변환
- Props와 State 관리
- 라이프사이클 메서드

**예상 학습 시간**: 4-6시간
**난이도**: ⭐⭐ (초급-중급)

### 2부: 고급 패턴
서비스, 의존성 주입, 라우팅 등 고급 패턴을 다룹니다.
- 서비스를 Custom Hooks로 변환
- 의존성 주입 대체 패턴
- 라우팅 시스템 마이그레이션
- 상태 관리 (NgRx → Redux Toolkit)

**예상 학습 시간**: 6-8시간
**난이도**: ⭐⭐⭐⭐ (고급)

### 3부: UI & 스타일링
템플릿, 스타일, 애니메이션 변환을 다룹니다.
- Template 문법을 JSX로 변환
- CSS 마이그레이션 전략
- 애니메이션 구현
- Material UI 마이그레이션

**예상 학습 시간**: 4-5시간
**난이도**: ⭐⭐⭐ (중급)

### 4부: 도구 및 환경
빌드 도구, 테스팅, 디버깅을 다룹니다.
- 빌드 도구 설정 (Webpack/Vite)
- 테스트 마이그레이션
- 디버깅 전략
- 성능 최적화

**예상 학습 시간**: 4-5시간
**난이도**: ⭐⭐⭐ (중급-고급)

### 5부: 실전 사례
실제 프로젝트 마이그레이션 경험을 공유합니다.
- 점진적 마이그레이션 전략
- 흔한 실수와 해결책
- 실제 케이스 스터디

**예상 학습 시간**: 3-4시간
**난이도**: ⭐⭐⭐⭐ (고급)

## 🎯 학습 목표

이 가이드를 완료하면:

- ✅ Angular의 모든 핵심 개념을 React로 변환할 수 있습니다
- ✅ 150개 이상의 실제 코드 예제를 활용할 수 있습니다
- ✅ 점진적 마이그레이션 전략을 수립할 수 있습니다
- ✅ 일반적인 문제를 예방하고 해결할 수 있습니다
- ✅ 현대적인 React 18+ 패턴을 적용할 수 있습니다
